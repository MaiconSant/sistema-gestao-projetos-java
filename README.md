# Sistema de Eventos em Java

Projeto desenvolvido para a disciplina de Programação de Soluções Computacionais.

## Descrição

Este sistema tem como objetivo realizar o cadastro e gerenciamento de eventos, permitindo que usuários visualizem, participem e acompanhem eventos em sua localidade. A aplicação foi desenvolvida em linguagem Java, utilizando execução em modo console e aplicando conceitos fundamentais de programação orientada a objetos.

## Funcionalidades

- Cadastro de eventos
- Listagem de eventos cadastrados
- Participação em eventos
- Cancelamento de participação
- Ordenação de eventos por data e horário
- Identificação de eventos passados, atuais e futuros
- Persistência de dados em arquivo (events.data)

## Estrutura do Projeto

- Usuario.java: Classe responsável pela representação do usuário
- Evento.java: Classe que define os atributos e comportamentos de um evento
- Sistema.java: Contém a lógica de negócio da aplicação
- Main.java: Classe principal responsável pela execução do sistema

## Conceitos Aplicados

- Programação Orientada a Objetos
- Encapsulamento
- Estruturas de dados (List)
- Manipulação de arquivos
- Organização e modularização de código

## Versionamento

Versão 1.0 – Estrutura inicial do sistema  
Versão 1.1 – Implementação das funcionalidades principais  
Versão 2.0 – Implementação de persistência em arquivo e melhorias no sistema  

## Execução

Para executar o sistema:

1. Abrir o projeto em uma IDE compatível com Java (IntelliJ, Eclipse ou NetBeans)
2. Executar a classe Main.java
3. Interagir com o sistema por meio do menu exibido no console
